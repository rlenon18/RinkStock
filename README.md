# RinkStock
This mock application has the ability to show Vendors, Products, Product Orders of a made-up hockey store. You can create receipts of product orders and then publish as PDF reports on pending shipments. 

# Overview
Rink Stock is a full-stack web application designed to demonstrate frontend, backend, and end-to-end system interactions. The project was used as a case study to explore testing strategies, API integration, and user flow validation

The application consists of:

- A frontend built with Angular
- A backend service handling API requests and data processing
- User interface with multiple views and routing
- API-driven data flow between frontend and backend
- Form handling and validation
- Modular and scalable structure

# Testing
Automated scripts were made to test via Cypress.
